# unix-shell-in-cpp

1. Ensure you have `cmake` installed locally

2. Run `./your_shell.sh` to run your program, which is implemented in
   `src/main.cpp`.

3. Run `cd build` to navigate to the build directory.
4. Run `./shell` to execute the shell program.
